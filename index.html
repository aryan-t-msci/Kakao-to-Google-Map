Coordinates are WGS84, the system Google, Kakao and GPS share. The Google link uses ?q=lat,lng, which pins the exact point and sidesteps Korean-address search quality entirely. The Lite path reads Kakao's place page through a public relay, so it's best-effort — fine for spot-checks. Doing thousands of rows? A Kakao REST key in a script is the batch tool, not a browser.

Full setup guide

How this tool is hosted, in five steps (A–E)

This page is already configured. Use this if you're re-hosting it, handing it to someone to maintain, or setting up a fresh copy. Everything here is free at spot-check volume.
A
Get a free Kakao JavaScript key
Go to developers.kakao.com, sign in, then My Application → Add an application. Open the app → App → Platform Key → JavaScript Key and copy the key value. (Not the REST or Admin key.)
B
Switch the Kakao Map feature on
In the same app: Product Settings → Kakao Map → Usage settings → ON (the toggle / 활성화 confirm). Without this, searches return nothing.
C
Put the key into the file
Open index.html in a text editor. Near the top of the script, set const BAKED_KEY = "your_key"; and save. A JavaScript key is domain-locked, so once step E is done it's safe to ship inside the page — it won't work from any other address.
D
Host it on GitHub Pages
Create a repo, upload index.html to the main branch. Then repo Settings → Pages → Build and deployment: Source Deploy from a branch, Branch main, folder / (root), Save. After a minute the site is live at https://USERNAME.github.io/REPO/ (the path is case-sensitive).
E
Register that address with Kakao, then open it
In the Kakao app: App → Platform Key → JavaScript Key → JavaScript SDK Domain, and add the origin only — https://USERNAME.github.io, with no /REPO/ path. Open your live URL: a green Full mode banner means it works. The setup panel's Test & save confirms it, or names which piece is still off.
Cost: a personal app's free Kakao quota (hundreds of thousands of calls/day) far exceeds spot-check use, and exceeding it returns an error rather than a charge — you're never billed unless you deliberately enable paid API. For thousands of rows, batch-geocode with a Kakao REST key in a script instead of a browser.
